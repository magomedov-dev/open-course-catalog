# 📚 Каталог курсов

Статичный сайт-каталог открытых курсов по Computer Science от MIT, Stanford, Harvard, UC Berkeley, CMU и других университетов. Курсы сгруппированы по категориям, с поиском и навигацией — без сборки, без зависимостей, только HTML/CSS/JS.

## Возможности

- Каталог курсов, сгруппированных по категориям (программирование, алгоритмы, системы, БД, сети, ML/AI, безопасность и др.)
- Поиск по коду курса, названию и университету в реальном времени
- Липкая боковая навигация с подсветкой активной категории при скролле
- Список курсов и категорий полностью вынесен в отдельный конфиг — правки не требуют трогать разметку или стили
- Ноль зависимостей: открывается как обычная статичная страница

## Источники

Все ссылки ведут на официальные страницы курсов соответствующих университетов. Каталог не хранит и не распространяет материалы курсов — только ссылки на них.

## Лицензия

Код репозитория можно использовать свободно (укажите здесь MIT или любую другую лицензию по вкусу). Материалы самих курсов принадлежат их университетам и правообладателям.
